Hamburger menu

Steps to build responsive menu

By Misha Vopat

HTML

Add content to the navigation menu - wrap that into <div> element.
It can include logo etc.
Specify <nav> section and provide navigation links.

Do not forget to attach stylesheet and JavaScript file.

CSS

At first style desktop layout.

Set up some basic styling for navigation wrapper (background colour, colour, padding...).
Style <a> element as well. As it is a link element, it has some default styling which we usually do not want to have for menu.
Use Flexbox -> display: flex. Because there is default flex direction = row, it compels items to align into one row.
Keep in mind that display:flex has an impact only on direct childer of the element.
Align navigation links to the right side by "margin-left: auto".
Remove toggle icon by set up "display:none".

Move on to mobile or tablet version.

Set up your breakpoints and media queries.
Then go through the same points as you went styling desktop layout.
Use Flexbox and align navigation menu into column ("flex-direction: column"). Hide navigation links but set up toggle icon visible again by using "display:flex" for the icon and "display:none" for navigation links. For this purpose, it is better to keep navigation links in separate class.

At this point it is time to connect JavaScript file.

JAVASCRIPT

Go back to your HTML code, insert the script and set up onclick="myFunction()" for toggle icon and navigation links class.
Compile myFunction() in JavaScript file.

function myFunction() {
const x = document.getElementById("myMenu");
if (x.className === "myLinks") {
x.className += " responsive";
} else {
x.className = "myLinks";
}
}

When the user clicks on the toggle icon, myFunction toggle between adding and removing the “responsive” class to “myLinks.”

THE LAST STEP

Add CSS rules for "responsive" class. Use "display: flex" and "flex-direction: column."
